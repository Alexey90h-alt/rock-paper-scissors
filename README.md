# 🎮 Игра "Камень, Ножницы, Бумага" на Python

Это мой учебный проект, в котором я реализовал классическую игру против компьютера. Программа использует случайный выбор и ведет учет счета в реальном времени.

---

### 🚀 Функционал:
* **Бесконечный цикл:** Играй столько, сколько захочешь.
* **Умный счетчик:** Программа запоминает твои победы и победы компьютера.
* **Обработка ошибок:** Если ты опечатаешься, программа вежливо попросит повторить ввод.
* **Регистронезависимость:** Можно писать "Камень" или "камень" — всё сработает!

### 🛠 Технологии:
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)
* Модуль `random` для логики компьютера.
* Циклы `while` и условные операторы `if/elif/else`.

### 📦 Как запустить игру (инструкция для новичков):

Не бойся, это проще, чем кажется! Тебе нужно сделать всего 3 шага:

1. **Установи Python** (если его еще нет):
   Скачай его с [официального сайта python.org](https://www.python.org/downloads/). При установке обязательно поставь галочку **"Add Python to PATH"** — это важно!

2. **Скачай код:**
   Нажми зеленую кнопку **Code** вверху этой страницы, выбери **Download ZIP** и распакуй архив. Или просто скопируй текст из файла `main.py` в новый файл у себя на компьютере.

3. **Запусти игру:**
   * Открой папку с файлом.
   * Зажми клавишу `Shift` на клавиатуре и кликни **правой кнопкой мыши** на пустом месте в папке.
   * Выбери пункт **"Открыть окно PowerShell здесь"** (или "Открыть в терминале").
   * В появившемся черном окне введи команду:
     ```bash
     python main.py
     ```
   * Нажми **Enter** и наслаждайся игрой! 🎮
